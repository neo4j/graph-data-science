# Domain hierarchy

This directory hosts the domain layer in the [standard stack](https://en.wikipedia.org/wiki/Multitier_architecture):

1) User interface
2) Applications
3) Domain
4) Integration

In this context we have the broad based functionality and services needed to do essential GDS things.
Think the different catalogs, metrics, progress tracking, stuff that is used in the [application/ algorithms layer](../applications/README.md) to build functionality.
